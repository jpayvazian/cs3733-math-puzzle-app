# cs3733MathPuzzleApp
## How to play 
```
Select a tile by clicking on it and a directional button to move.
The four directions coorespond to certain mathematical operations on the tile's values:

Up is multiply
Down is divide (destination tile / selected tile) Valid only if result is whole number
Left is subtract (destination tile - selected tile) Valid only if result is positive
Right is addition

```
## Winning condition
```
Only tile remaining is the center.
If there are no more valid moves, game is over. Can reset puzzle anytime.
```
